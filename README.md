
Main Features

Task Management (CRUD): Create, edit, view, and delete folders or tasks.

Assignment and Tracking: Assign tasks to team members and monitor their progress.

Search and Filtering: Quickly find folders or tasks by type, priority, or date.

Email Notifications: Automatically send emails to remind deadlines or notify collaborators of new tasks.

Event Calendar: View all tasks and deadlines on an interactive calendar.

Modern User Interface: Built with React for a smooth and responsive user experience.

Secure REST API: Communicate with the backend via Spring Boot and a MySQL database.

Installation and Running

Backend
cd backend
mvn spring-boot:run

Frontend
cd frontend
npm install
npm run dev
