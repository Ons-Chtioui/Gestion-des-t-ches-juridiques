Fonctionnalités principales

Gestion des tâches (CRUD) : créer, modifier, consulter et supprimer des dossiers ou tâches.

Assignation et suivi : attribuer les tâches aux collaborateurs et suivre leur avancement.

Recherche et filtrage : retrouver rapidement les dossiers par type, priorité ou date.

Notifications par email : envoyer automatiquement des emails pour rappeler les échéances ou informer les collaborateurs des nouvelles tâches.

Calendrier des événements : visualiser toutes les tâches et échéances sur un calendrier interactif.

Interface utilisateur moderne : conçue avec React pour une expérience fluide et responsive.

API REST sécurisée : communication avec le backend via Spring Boot et une base de données  MySQL.

Installation et exécution
Backend
cd backend
mvn spring-boot:run

Frontend
cd frontend
npm install
npm run dev
